# **App Name**: ProducTrack

## Core Features:

- Machine Registration: Users can register new machines by inputting essential details like model, ID, capacity, and current status.
- Production Data Logging: Allows operators to log daily production outputs, downtime events, and quality metrics for each machine.
- Real-time Dashboard: Provides a high-level overview dashboard displaying critical production KPIs, machine statuses (running, idle, fault), and live output graphs.
- Basic Production Reporting: Generate and view simple reports based on production data, such as daily/weekly output summaries and machine utilization rates.
- AI Production Insights: An AI tool that analyzes logged production data to identify potential bottlenecks, suggest efficiency improvements, or flag anomalies in machine performance.

## Style Guidelines:

- Primary color: A sophisticated, deep blue (#2671D9) symbolizing precision and technology. This will be used for interactive elements and key headers.
- Background color: A very light, almost white blue (#F5FAFF) providing a clean, airy canvas that is easy on the eyes and highlights content.
- Accent color: A vibrant aqua (#47EBDF) to draw attention to important alerts, call-to-action buttons, or status indicators, creating contrast and dynamism.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring excellent readability for data and labels.
- Clean, linear, and technically-inspired icons will be used throughout the application to represent machines, production processes, and data visualizations, maintaining a consistent professional look.
- Modular card-based layouts will be employed for displaying machine details and production metrics, allowing users to easily scan and comprehend information at a glance. Grids and responsive design ensure adaptability across devices.
- Subtle and functional animations, such as smooth transitions for dashboard updates and filter changes, will enhance user experience without causing distraction, reflecting the app's focus on efficiency.
